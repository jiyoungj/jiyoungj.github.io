About the database

This database provides a collection of speech recordings and articulatory data produced under controlled experimental conditions. The database includes annotated speech materials, metadata about speakers, and documentation describing the experimental design and data structure.

The goal of the database is to facilitate reproducible research and investigation of phonetic and articulatory realization in Korean.

How to navigate

Detailed documentation describing the structure and organization of the database can be found in the Description tab in the menu bar.
