데이터베이스 설명

1. 데이터 수집

본 섹션에서는 데이터의 수집 과정에서 사용된 녹음 절차, 실험 설계, 장비 및 기타 세팅에 대해 설명합니다.

1.1. 사용 장비

조음 데이터(Articulatory data): Carstens AG501
음향 데이터(Acoustic data): 마이크 및 TASCAM

1.2. 실험 환경

• 서울 한양 음성∙언어인지과학연구소 내 실험실
• Tascam HC-P2 디지털 레코더 및 SHURE KSN44 콘덴서 마이크
• 총 8개의 센서 코일 부착 (i.e., tongue tip, body, dorsum, lower/upper incisors, lower/upper lips, naison)

2. 데이터 내용

본 섹션에서는 데이터베이스 수집 당시 사용된 자료를 설명합니다. 여기에는 읽기 지문, 음절 목록, 또는 실험 자극(예: 발화 속도 조절) 등이 포함됩니다.

3. 데이터 형식

본 데이터베이스에는 다음과 같은 자료가 포함되어 있습니다:

• 음성 녹음 파일 (WAV 형식)
• 조음 데이터 (MAT 형식)
• 주석이 포함된 TextGrid 파일
• 각 화자에 대한 메타데이터 파일 (PDF 형식)

4. 화자 정보
